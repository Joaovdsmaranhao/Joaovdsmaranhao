## 👤 Sobre Mim

Sou desenvolvedor e estudante de Ciência da Computação no Rio de Janeiro. Crio soluções que resolvem problemas reais — desde aplicações desktop e sistemas web até automações e integrações entre plataformas.

- 🤖 Automação de processos com Python (Selenium, Pandas, Matplotlib)
- ⚡ Stack principal: JavaScript, Node.js, Electron, React e desenvolvimento web
- 🔗 Integrações e fluxos automatizados com n8n
- 🎯 Foco em código limpo, bem estruturado e orientado a resultados
